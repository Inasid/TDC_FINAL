Lucknow Lattice is a generative sub-identity system for the city, using dynamic display typography and patterned backdrops driven by an interactive tool. The system maps four architectural motifs—Nawabi jali, Indo-Grecian floral, Colonial star, and Contemporary bloom—to specific neighborhoods based on historical density. Users control pattern mixing to reflect how architectural styles overlap across districts
